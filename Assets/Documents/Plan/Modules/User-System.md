# 用户系统

## 概述

管理用户身份与学习数据。`UserData` 包含登录凭证、实验进度、成绩记录，通过 JSON 持久化。

## 当前状态

**优先级低**，在核心功能（场景管线、漫游、交互、步骤系统）完成后再实施。

## 计划功能

- **多用户切换**：本地存档槽位模式（非联网账号密码），存在同一 `UserData.json` 中用 key 区分
- **进度保存**：按 Task 粒度保存当前实验进度
- **成绩记录**：Test 小测验和 Training 仿真实验得分
- **数据隔离**：多用户各自独立存档
